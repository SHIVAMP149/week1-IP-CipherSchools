THESE ALL ARE GLOBAL ATRRIBUTE

CLASS AND ID ATTRIBUTE WILL BE USED IN CSS AND JAVASCRIPT.

--CLASS ATTRIBUTE--

SAME CLASS ATTTIBUTE CAN BE USED IN SO MANY ELEMENTS.

--ID ATTRIBUTE--

BUT IN ID IT ONLY BE SPECIFIED TO ONE ELEMENT.

--LANGUAGE ATTRIBUTE--

LANGUAE ATTRIBUTE CHANGES LANGUAGE OF THE CONTENT.

--DIRECTION ATTRIBUTE--

IT CHANGES DIRECTION FROM LEFT TO RIGHT OR RIGHT TO LEFT.

--CONTENTEDITABLE ATTRIBUTE--

IF CONTENTEDITABLE VALUE IS SET ON TRUE THEN WE CAN EDIT IN THAT ELEMENT.
